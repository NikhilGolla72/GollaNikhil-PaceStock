üöÄ Quick Start

Note: FluxBack is a C++ CLI tool that runs locally on your machine. Clone the repository to get started!

1. Clone the Repository

git clone https://github.com/NikhilGolla72/GollaNikhil-PaceStock.git cd GollaNikhil-PaceStock

2. Build the Project

Windows (PowerShell):

# Install CMake first: winget install Kitware.CMake # Then build: .\build.ps1

Linux/macOS:

mkdir build && cd build cmake .. -DCMAKE_BUILD_TYPE=Release cmake --build . --config Release

3. Run a Backtest

Windows:

cd build\Release .\fluxback.exe run --strategy ..\..\config\sma_demo.yaml --data ..\..\demo\aapl_sample.csv --out ..\..\results\demo.json

Linux/macOS:

cd build ./fluxback run --strategy ../config/sma_demo.yaml --data ../demo/aapl_sample.csv --out ../results/demo.json

Or use the quick demo script:

# Windows .\run_demo.ps1 # Linux/macOS ./run_demo.sh
